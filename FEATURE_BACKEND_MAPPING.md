# Frontend to Backend Feature Mapping

This document provides a comprehensive overview of the integration between the Job Portal Pro frontend applications (Job Seeker, Company, and Admin) and the provided `Job portal backend`.

## Summary

The frontend has been fully connected to the backend for all core functionalities. Mock data has been removed for authentication, job/applicant management, and user/company profiles. The API client in `packages/api-client` now communicates exclusively with the backend endpoints via a proxy to `http://localhost:3000`.
